Socialmoodia

Socialmoodia is a full-stack project designed to simulate a social media app where users can only post and comment with emojis. The back-end was created and maintained using Sequelize (or JawsDB on Heroku) in conjunction with node.js, express.js. Users could be created and authenticated using cookies and bcrypt. The front end served data through use of handlebars to dynamically display webpages.

Built with:

  • Node.js
  • Sequelize
  • Express
  • Bcrypt
  • Handlebars

Off the Hook

Off the Hook is a Front-end project that calls upon other APIs to update and display excuses and memes that the APIs can respond with. The project also utilized local storage to keep an active history of responses.

Built with:

  • HTML, CSS
  • JavaScript
  • Excuser, ImgFlip APIs
  • UIKit
